## 需求 1: 简化模型配置

### 现状
- 界面上有"模型配置"面板，需要手动选择检测模型和分类模型的路径
- 有"参数设置"面板，需要手动调节阈值

### 目标
- **去掉 UI 上的模型配置和参数设置**
- 改为**自动加载**：程序启动时扫描以下目录：
  - `models/YOLO/` —— 取第一个文件作为检测模型
  - `models/ResNet/` —— 取第一个文件作为分类模型
- 参数使用代码中写死的默认值：
  - 检测置信度: 0.25
  - 检测尺寸: 768
  - 大框比例阈值: 0.30
  - 大框分类阈值: 0.55
  - 分类尺寸: 224

### 状态
- [x] 已确认: 文件夹结构为 `models/YOLO/` 和 `models/ResNet/`
- [x] 已确认: 各文件夹放一个模型文件即可

## 需求 2: 输出能力增强

### 目标
实现**"怎么输入怎么输出"**：
- 输入图片 → 输出标注后的图片
- 输入视频 → 输出标注后的视频
- 输入摄像头 → 可选多种输出方式

### 输出选项
| 输入类型 | 输出选项 |
|---------|---------|
| 单张图片 | 输出到指定目录，保留原文件名 |
| 文件夹图片 | 批量输出到指定目录 |
| 单个视频 | 输出为标注后的视频文件 |
| 文件夹视频 | 批量输出 |
| 摄像头流 | ① 不保存（仅预览）<br>② 周期性截图保存<br>③ 录制为 MKV 视频文件 |

### 状态
- [ ] UI 设计
- [ ] 后端实现
